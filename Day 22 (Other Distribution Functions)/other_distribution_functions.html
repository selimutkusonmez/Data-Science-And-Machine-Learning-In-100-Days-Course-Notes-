Poisson Distribution

Definition : Poisson Distribution is used to calculate the possiblity of the event that takes place in a certain time or place

Example : A call centers daily call number

$$ P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!} $$

Poisson Distribution CDF

$$ F(k) = P(X \le k) = \sum_{i=0}^{k} \frac{\lambda^i e^{-\lambda}}{i!} $$

Normal Distribution (Continuous Variables)

Definition : The Normal Distribution is the most common distribution used to describe the probabilities of various phenomena./p>

Example : Human height distribution/p>

$$ f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $$

Normal Distribution CDF

$$F(x) = \frac{1}{\sqrt{2\pi}\sigma} \int_{-\infty}^{x} e^{-\frac{(t-\mu)^2}{2\sigma^2}} dt$$

Standard Normal Distribution

Definition : Standard Normal Deviation is a special type of Normal Distribution and it has these features.

  • Mean (µ) : 0
  • Standard Deviation (σ) : 1

Example :

$$f(z) = \frac{1}{\sqrt{2\pi}} e^{-\frac{z^2}{2}}$$

Z Score

Definition : A Z-score shows us how many standard deviations a data point is away from the mean.

$$ Z = \frac{X − µ}{σ} $$

Uniform Distribution

Definition : Uniform Distribution is a probability distribution where every value within a spesific range has the same proabability

Example : Lottery

$$f(x) = \begin{cases} \frac{1}{b-a}, & a \le x \le b \\ 0, & \text{other cases} \end{cases}$$

Uniform Distribution CDF

$$F(x) = \begin{cases} 0, & x < a \\ \frac{x-a}{b-a}, & a \le x \le b \\ 1, & x > b \end{cases}$$

Log Normal Distribution

Definition : Log-Normal Distribution is where the logarithm of the values is normally distributed

Example : Income Distribution

$$f(x) = \frac{1}{x\sigma\sqrt{2\pi}} e^{-\frac{(\ln x - \mu)^2}{2\sigma^2}} , \quad x > 0$$

Log-Normal Distribution CDF

$$ F(x) = P(X \le x) = \frac{1}{2}\left(1 + \operatorname{erf}\left(\frac{\ln x - \mu}{\sigma\sqrt{2}}\right)\right) $$

Pareto Distribution

Definition : A Pareto Distribution is a skewed, heavy-tailed probability distribution often used to describe the 80/20 rule, where a small percentage of causes produces a large percentage of effects.

Example : City Populations

$$f(x) = \frac{\alpha x_m^\alpha}{x^{\alpha+1}}, \quad x \ge x_m$$

Pareto Distribution CDF

$$ F(x) = 1 - \left(\frac{x_m}{x}\right)^\alpha, \quad x \ge x_m $$
