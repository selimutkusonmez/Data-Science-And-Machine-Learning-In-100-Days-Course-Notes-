Introduction to SQL

How To Extract Data

  • SELECT * FROM table_name - Select everything in table_name
    • SELECT * FROM my_table
  • SELECT column_name from (table_name) - Select column or columns in table_name
    • SELECT name from my_table
  • SELECT * FROM table_name WHERE column_name = value - Select everything from table where column_name = value
    • SELECT * FROM my_table WHERE name = 'Jane Doe'
  • SELECT * FROM table_name WHERE column_name (<,>) value - Select everything from table where column_name (<,>) value
    • SELECT * FROM my_table WHERE age > 20
  • SELECT * FROM table_name WHERE column_name LIKE value(example = 'S% starts with s','%S endswith s') - Select everything from table_name where column is similar to the value
    • SELECT * FROM my_table WHERE name LIKE 'J%'

How To Insert Data

  • INSERT INTO table_name (column_names) VALUES (values)
    • INSERT INTO my_table (name,age) VALUES ('Jane Doe',20)

How To Update Data

  • UPDATE (table_name) SET column_name = value WHERE column_name = value
    • UPDATE my_table SET age = 20 WHERE name = 'Jane Doe'

How To Remove Data

  • DELETE FROM table_name WHERE column_name = value
    • DELETE FROM my_table WHERE name = 'Jane Doe'

How To Create Table

  • CREATE TABLE IF NOT EXISTS table_name (column_name (column_value_type column_features))
    • CREATE TABLE IF NOT EXISTS data (id INTEGER PRIMARY KEY AUTO INCREMENT, name VARCHAR(20), surname VARCHAR(20), age INTEGER)
