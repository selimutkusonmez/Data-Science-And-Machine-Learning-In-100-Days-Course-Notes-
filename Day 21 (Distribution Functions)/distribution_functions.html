What is PMF (Probability Mass Function)

Definition : PMF is a function that shows the probability of the value that discrete random variables can take

Example : Here's an example of a call center's daily call numbers

$$ P(X = x) = f(x) $$

What is PDF (Probability Density Function)

Definition : PDF is a function that shows the probability of the value that continuous random variables can take

Example : Students height graph of a high school

$$ P(a ≤ X ≤ b) = \int_{a}^{b} f(x)dx $$
$$ f(x) ≥ 0 $$

What is CDF (Cumulative Distribution Function)

Definition : CDF shows us the total possibility of a value range that a random variable can have

Example : When we roll a die the total possibility of every possible event

$$ P(X ≤ 1) = 1/6 $$ $$ P(X ≤ 2) = 2/6 $$ $$ P(X ≤ 3) = 3/6 $$ $$ P(X ≤ 4) = 4/6 $$ $$ P(X ≤ 5) = 5/6 $$ $$ P(X ≤ 6) = 6/6 $$

Bernoulli Distribution

Definition : Bernoulli Distribution is a basic probability distribution that shows us events that have only 2 possible outcomes

Example : The possibilities of heads and tailf when tossing a coin

$$ P(X=k) = \begin{cases} p, & k=1 \\ 1-p, & k=0 \end{cases} $$

Binomial Distribution

Definition : The Binomial Distribution describes the number of successes in n independent Bernoulli trials with the same probability.

Example : Tossing a coin 10 times and landing on heads

$$ P(X=k) = \binom{n}{k} p^k (1-p)^{n-k} $$
